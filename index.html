ABOUT APIKERNEL API

With this site, we hope to provide the World of Warcraft developer community with access to game data in a secure and reliable way. We believe that is time to provide the World of Warcraft private community the tools to create amazing third-party tools for the server they play on. We want this API to meet the same high standards as our APIKernel products, so we'll be iterating and evolving as we hear your feedback and suggestions (so share 'em all).
